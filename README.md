# Lealtix Frontend

Lealtix es una aplicación web enfocada en ayudar a pequeños negocios como restaurantes, tiendas y boutiques a **atraer y retener clientes** mediante estrategias de lealtad y marketing personalizado.  
Este repositorio contiene el **frontend** desarrollado en **Angular** con **PrimeNG**, diseñado para ser responsivo y escalable.

---

## 🚀 Características principales

- 🎯 **Página de marketing** atractiva para presentar el producto.
- 📝 **Registro de tenant** para nuevos negocios.
- 📱 **Diseño responsivo** adaptable a dispositivos móviles y escritorio.
- 🎨 **Interfaz moderna** utilizando PrimeNG.
- 🔄 **Arquitectura modular** para facilitar el crecimiento de la aplicación.

---

## 📦 Tecnologías utilizadas

- [Angular](https://angular.io/) (versión X)
- [PrimeNG](https://www.primefaces.org/primeng/)
- [Node.js](https://nodejs.org/) (para dependencias y compilación)
- [TypeScript](https://www.typescriptlang.org/)

---

## ⚙️ Instalación y ejecución local

Sigue estos pasos para clonar y correr el proyecto en tu máquina local:

```bash
# 1. Clonar el repositorio
git clone https://github.com/KikeGitHub/lealtix-FE.git

# 2. Entrar al directorio del proyecto
cd lealtix-FE

# 3. Instalar dependencias
npm install

# 4. Levantar el servidor de desarrollo
ng serve

# 5. Abrir en navegador
http://localhost:4200
